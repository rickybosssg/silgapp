## 📐 Spécifications détaillées

### Icône (512x512 px)
```
Format : PNG 32-bit
Couleur : Supporte la transparence
Ombres : Interdites
Texte : Interdit
Formes géométriques : Éviter
Contenu : Doit représenter SILGAPP (logo, livraison, coursier)
```

### Feature Graphic (1024x500 px)
```
Format : PNG ou JPEG
Couleur : RVB ou CMJN
Texte : Autorisé mais minimal
Logo : Peut inclure le logo SILGAPP
Message : Doit communiquer l'essence de l'app
```

### Screenshots
```
Résolution : 1080x1920 px (recommandé)
Format : PNG ou JPEG
Orientation : Portrait ou paysage
Contenu : Montrer les fonctionnalités clés
Annotations : Autorisées (flèches, textes explicatifs)
```

## 🎨 Suggestions de design SILGAPP 2

**Éléments recommandés :**
- Logo SILGAPP (rouge #FF0000)
- Icône de livraison (colis, moto, localisation)
- Fond blanc ou dégradé rouge/orange
- Style moderne et épuré

**Exemple de composition :**
- Fond : Dégradé rouge (#FF0000 → #FF6B6B)
